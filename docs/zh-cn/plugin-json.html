JSON 插件

使用 seajs.config 激活 JSON 插件： 

seajs.config({
  preload: ['plugin-json']
});

然后通过 require 加载：

a.json:

{
  "name": "Frank",
  "age": 30
}

main.js:

define(function(require) {
  var json = require('./a.json');
  // do sth.
});

除了 .json 后缀, 还可以使用 json! 前缀来指明 JSON 文件。
